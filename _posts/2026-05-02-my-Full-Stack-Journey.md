# My AWS Journey 🚀

Hi, I'm Anchal! This is my first blog post and I want to share 
what I've been building and learning on AWS.

## What I've Built So Far

### ⚡ ATS³L — Automated S3 Tagger via Lambda
My first real project! I built a serverless system where uploading 
a JSON file to S3 automatically triggers Lambda to tag the bucket.
No manual steps — fully automated with Python and Boto3.

### 🖥️ EC2 Instance
I launched a Ubuntu backend server on EC2 and configured VPC, 
subnets, security groups and SSH key pairs from scratch.

## What I Learned
- Serverless architecture is powerful and fun to build
- IAM permissions are confusing at first but make sense once you break them down
- CloudWatch logs are your best friend when debugging Lambda
- Every error teaches you something new

## What's Next
- Deepen my AWS knowledge
- Understand how **APIs connect** frontend and backend services
- Start building Full Stack projects
