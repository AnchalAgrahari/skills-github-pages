## What I've Build on AWS

### ATS³L — Automated Tagger in S3 via Lambda
My first open-source project - a fully serverless, event-driven system that:
- Automatically triggers **AWS Lambda** when a JSON file is uploaded to **S3**
- Dynamically reads and parses JSON to apply tags to S3 buckests
- Build with **Python, Boto3, IAM Roles, S# Event Notification & CloudWatch**
- Designed to be reusable, scalable, and hands-free

### EC2 Instance Deployment - Backend Server
Launched a production level Ubuntu backend server on AWS EC2:
- Solved the core problem of **24/7 app availability** (vs local system limits)
- Configured **VPC, Subnet, Security groups, & SSH Key Pair(RSA/ .pem**
- Set up **Inbound Rules** for TCP/SSH access and public IP assignment
- Attached **EBS storage (gp3)** for application and OS data
- Instance Type: **ta3.micro** (AWS Free Tier eligible)
